# Task 1 — Number Analysis App

Location:
task1/number_analysis.dart

This program works with a list of numbers and performs several types of analysis.

The program:

• Finds the maximum number  
• Finds the minimum number  
• Calculates the sum of all numbers  
• Calculates the average value  
• Counts how many numbers are negative  

The functions are written manually using loops rather than built-in Dart methods.  
This helps demonstrate understanding of logic and iteration.



# Task 2 — Async Calculator App

Location:
task2/calculator_app.dart

This program demonstrates object-oriented programming and asynchronous programming.

The Calculator class includes methods for:

• Addition  
• Subtraction  
• Multiplication  
• Division with error handling  

The divide() method throws an error if the user attempts to divide by zero.

The program also includes asynchronous calculations using:

async  
await  
Future.delayed()

This simulates a real-world delay similar to waiting for a server or database.

The calculator also stores a history of previous calculations.
